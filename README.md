# YellowLight Hub

A full-stack ecommerce platform for a dropshipping business in Bangladesh.

## Tech Stack
- **Frontend:** Next.js 14 (App Router)
- **Backend:** Express.js
- **Database:** Supabase (PostgreSQL)
- **Styling:** Tailwind CSS

## Project Structure
- `/frontend`: Next.js application
- `/backend`: Express.js server

## Getting Started

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`

### Backend
1. `cd backend`
2. `npm install`
3. `npm start`
