# 🌟 YellowLight Hub

[![Built with Next.js](https://img.shields.io/badge/Next.js-14-black?style=for-the-badge&logo=next.js)](https://nextjs.org/)
[![Powered by Supabase](https://img.shields.io/badge/Powered%20by-Supabase-emerald?style=for-the-badge&logo=supabase)](https://supabase.com/)
[![Styled with Tailwind](https://img.shields.io/badge/Styled%20with-Tailwind-38B2AC?style=for-the-badge&logo=tailwind-css)](https://tailwindcss.com/)

A premium, full-stack e-commerce marketplace engineered for the modern dropshipping landscape in Bangladesh. **YellowLight Hub** combines a high-performance Next.js frontend with a robust Express.js backend, all seamlessly integrated with Supabase for real-time data and authentication.

---

## 🚀 Experience the Performance

- **Optimized for Speed**: Utilizing `next/image` for lightning-fast loads and `TanStack Query` for intelligent caching.
- **SEO Masterclass**: Dynamic sitemaps, robots.txt, and JSON-LD structured data are baked in.
- **Premium UI**: A sleek, custom-designed interface with smooth transitions, skeleton states, and responsive layouts.
- **Full-Stack Logic**: Complex inventory management, digital asset delivery, and order lifecycle tracking.

---

## 🛠️ Architecture Overview

YellowLight Hub is structured as a **Monorepo** for easier management and deployment:

```text
YellowLight_Hub/
├── frontend/         # Next.js 14 Web Application
│   └── src/          # React components, stores, & logic
└── backend/          # Express.js REST API
    └── src/          # Specialized routes & controllers
```

## 🏁 Quick Start

### 1. Prerequisite
Ensure you have **Node.js 18+** installed on your system.

### 2. Configuration
Create `.env` files in both `frontend` and `backend` directories using the provided `.env.example` templates.

### 3. Installation & Launch

#### 🖥️ Frontend
```bash
cd frontend
npm install
npm run dev
```

#### ⚙️ Backend
```bash
cd backend
npm install
npm run dev
```
